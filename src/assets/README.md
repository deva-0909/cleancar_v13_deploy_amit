# Assets

This directory contains static assets for the application.

## Structure

- **/images** - Image files (PNG, JPG, SVG)
- **/icons** - Icon files and custom icons
- **/fonts** - Custom font files
- **/logos** - Company and brand logos

## Guidelines

- Optimize all images before adding them
- Use SVG for icons when possible
- Name files descriptively using kebab-case
- Add README files in subdirectories for organization

## Naming Convention

- Use lowercase with hyphens: `employee-avatar.png`
- Include size in filename if multiple sizes: `logo-small.svg`, `logo-large.svg`
- Group related assets in subdirectories
