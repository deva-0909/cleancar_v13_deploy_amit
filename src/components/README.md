# Components

This directory contains reusable business components that are specific to the car washing service application.

## Structure

- **/shared** - Shared business components used across modules
- **/domain** - Domain-specific components (e.g., EmployeeCard, CustomerCard, VehicleCard)
- **/features** - Feature-specific components

## Guidelines

- Components here should use design system components as building blocks
- Focus on business logic and domain-specific functionality
- Keep components reusable and well-documented
- Use TypeScript interfaces for all props

## Examples

- `EmployeeCard` - Displays employee information
- `CustomerCard` - Displays customer details
- `VehicleCard` - Shows vehicle information
- `AppointmentCard` - Appointment details
